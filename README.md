# OpenEars Website Mockup

This project is a website mockup for **OpenEars**, an application designed to help users visualize and track their resources, team, and timeline.

## Project Overview

The website serves as a landing page and demonstration for the OpenEars app, showcasing its key features, the team behind it, and the development roadmap.

## Features

- **Home Page**:
  - **Hero Section**: Introduction to OpenEars with a call to action.
  - **Features Section**: Highlights key functionalities of the app.
  - **Team Section**: Introduces the team members.
  - **Timeline Section**: Displays the project's roadmap and milestones.
- **Demo Page**: A demonstration page for the app's capabilities.
- **Navigation**: Fully functional Header and Footer for easy navigation.

## Tech Stack

- **Framework**: React
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **Styling**: CSS

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
